# iRipple Website (Next.js)

## Run locally
1. Install dependencies:
   - `npm install`
2. Start dev server:
   - `npm run dev`
3. Open:
   - `http://localhost:3000`

## Notes
- All page copy is generated from `content/siteContent.js` (from the provided spreadsheet).
- TailwindCSS is configured in `tailwind.config.js`.
